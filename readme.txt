Website: Artsy - Art Portfolio Website

About the Website: 
Artsy is a colorful and interactive art portfolio website that showcases various artworks and multimedia elements that I have created so far. It has a welcoming interface with a navigation menu, image gallery, multimedia support, and a contact form. This website is aimed at creating an online space for my artistic side to share work.

Tags Used: 

HTML Tags:

<!DOCTYPE html> Defines the document type and version of HTML.
<html lang="en"> The root element of the document with language set to English.
<head>Contains meta information about the document such as title and linked stylesheets.
<meta charset="UTF-8">: Specifies the character encoding for the document (UTF-8).
<meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures responsive design by controlling the viewport settings.
<link rel="stylesheet" href="style.css">: Links the external CSS file for styling.
<link rel="icon" type="image/png" href="favicon.png">: Links to a favicon image for the website.
<title>: Sets the title of the webpage.
<body>: Contains the visible content of the webpage.
<header>: Defines the header section of the webpage, containing the introductory content and navigation.
<h1>: Heading tag for the website title ("Hello Art Lover!!!").
<p>: Paragraph tag used to display the introductory text.
<hr>: Horizontal line used to separate sections.
<nav>: Defines the navigation section with links to different parts of the webpage.
<menu>: Used to group navigation links inside the `nav` element.
<ul>: Unordered list used for the navigation links.
<li>: List item tag used inside the unordered list for each navigation link.
<a href="#home" class="button">: Anchor tag used for navigation links, with an applied button style.
<section>: Defines sections in the webpage for the gallery, categories, details, and contact.
<div>: Generic container element, used for grouping gallery and multimedia content.
<a href="vase.png" target="_blank">: Anchor tag to link images in the gallery to open in a new tab.
<img src="vase.png" alt="Vase Painting using Gouache colors">: Embeds an image in the webpage with a description for accessibility.
<audio controls>: Embeds an audio player with playback controls.
<video width="320" height="240" controls>: Embeds a video player with playback controls.
<table>: Defines a table for displaying different art categories with images.
<tr>, <th>, <td>: Table row (`<tr>`), table header (`<th>`), and table data (`<td>`) tags used to structure the table.
<details>: Creates a collapsible content section for art commissions.
<summary>: Visible heading for the `<details>` element, providing the option to expand or collapse the section.
<form>: Defines the contact form for user inputs.
<label for="name">: Label tag used to describe input fields in the form.
<input type="text" id="name" name="name">: Text input field for the user's name.
<textarea>: Multiline input field for longer text messages.
<footer>: Defines the footer section containing copyright information.

CSS Tags:

h1: Styles for the main header, including color, background color, text alignment, padding, and font size.
p: Styles for paragraphs, including text color, alignment, and font size.
nav: Styles for the navigation bar, including background image, padding, height, and content alignment.
nav ul: Styles for the navigation list, including list styling, layout, and spacing.
nav ul li: Styles for list items in the navigation, including margins.
nav ul li a: Styles for navigation links, including padding, font size, and hover effects.
section: Styles for webpage sections, including margin, padding, background color, and border radius.
table: Styles for the table, including border, padding, and alignment of table elements.
lightbox-gallery: Styles for the image gallery layout, including display settings, gap, and padding.
details-summary: Styles for the summary text inside the collapsible section.
order-message: Custom style for the message displayed in the 'Details' section.
